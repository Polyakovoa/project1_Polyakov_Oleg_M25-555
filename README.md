# Текстовая игра "Лабиринт"

Исследуйте таинственный лабиринт, решайте загадки, собирайте предметы и находите сокровище!

## Описание игры

Вы просыпаетесь в древнем лабиринте, полном загадок и опасностей. Ваша цель - найти путь к сокровищнице и открыть сундук с сокровищами. По пути вам предстоит:

- Исследовать различные комнаты
- Решать головоломки и загадки
- Собирать полезные предметы
- Избегать ловушек
- Использовать инвентарь для преодоления препятствий

# Клонируйте репозиторий

cd project1_Polyakov_Oleg_M25-555
